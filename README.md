# CS360
The goal of this app is to have an event tracker where you can login, add events, after adding an event then you can edit, update, or delete the event if needed, and enable SMS notifications to be reminded of upcoming events.
We need multiple classes to connect a database housing the events and details, a class that adds events to the database, edit database entries, XML’s to format the login, add event screen, and enable SMS notifications. The users were kept in mind by making all screens a uniform format keeping the spacing as similar as possible.
I broke this down into smaller chunks getting the XML design uniform first and then adding the backend support to make sure those screens are functional. Just making sure the connectivity between classes and the XML’s had no errors.
Checking for errors typos and asking friends and family to look over for functional or just typos.
The most challenging part was the connectivity between classes and XML’s it took looking into researching on YouTube, reaching out to friends and family who have experience coding asking if my logic makes sense and having them check for typo’s or any technical errors
